## How to use

Every prompt is labeled for the mode it works best in:
- **[Composer]** — agent mode (Cmd-I), multi-file edits
- **[Chat]** — right-side panel, single-turn
- **[Inline]** — Cmd-K edits inside a file
- **[.cursorrules]** — paste into your rules file; applies to every prompt

Read the legend at the top of PROMPTS.md. Pick a mode, pick a prompt, replace placeholders, go.
