# Launch-Day Playbook (Pack-01)

Pick a Tuesday or Wednesday. Avoid Monday (people catching up) and Thu/Fri (weekend drift). Avoid launching during major tech events unless the pack relates.

Do this once, then reuse the rhythm for pack-02+.

---

## T-1 day (night before)

- [ ] Product is live on Gumroad as a draft. Finalize: title, subtitle, price $29, discount code `LAUNCH17` (active 72hr), description from `marketing/pack-01-gumroad-listing.md`, cover image uploaded.
- [ ] Click "Publish" so the product is live. Test-buy with your own email to confirm: checkout works, delivery email arrives, downloaded file opens.
- [ ] Schedule the X thread via Typefully for **10:00 AM local**. Each tweet 90 seconds apart.
- [ ] Draft Reddit post in notes — don't schedule, you'll post it manually (needs you logged in).
- [ ] Draft LinkedIn post. Schedule for **1:00 PM local**. First comment pre-written with the link.
- [ ] Queue Threads post 1 for **11:30 AM local**, post 2 for **5:00 PM local**.

---

## Launch day — minute-by-minute

### 9:45 AM — Final check (5 min)
- [ ] Gumroad link loads in an incognito tab and shows correct price + cover
- [ ] Discount code `LAUNCH17` still active and correct price applies
- [ ] Your scheduled X thread is still queued (Typefully)

### 10:00 AM — X thread fires (automatic)
- [ ] Within 15 min: reply to the thread with a bonus — one more prompt not in the main thread. Keeps it alive in feed.
- [ ] Retweet the first tweet from any alt accounts you have (legitimately, not fake).

### 11:00 AM — Reddit (30 min work, then walk away)
- [ ] Post to **r/ChatGPTPromptGenius** first — it's the most forgiving. Title + body from `marketing/pack-01-launch-content.md`.
- [ ] Wait 20 min. If it lands (>5 upvotes, no mod removal), cross-post to **r/ClaudeAI** with the title modified slightly so it doesn't look spammy.
- [ ] Do NOT post to more than 2 subs today. Reddit spam-detection is fast.

### 11:30 AM — Threads post 1 fires (automatic)

### 1:00 PM — LinkedIn post fires (automatic)
- [ ] Immediately post the link as the first comment (do this yourself — scheduled comments don't always fire reliably).
- [ ] Reply to every comment in the first 2 hours. Algorithm weights early engagement heavily.

### 2:00 – 5:00 PM — Comment reply shift (30 min of actual work, spread across 3 hrs)
- [ ] Check X replies, reply to everything. Keep replies short (1–2 sentences). If a question is juicy, write a mini-reply-thread.
- [ ] Check Reddit comments. Answer technical questions fully — these convert directly to sales.
- [ ] Check LinkedIn comments. Professional register.

### 5:00 PM — Threads post 2 fires (automatic)

### 6:00 PM — First checkpoint (5 min)
- [ ] Open Gumroad analytics. Note: views, sales, conversion rate.
- [ ] If sales = 0 after 8 hours: not alarming. Month 1 without audience can convert slow. Keep going.
- [ ] If conversion > 3%: your listing works; push more traffic.
- [ ] If conversion < 1% despite traffic: listing copy or price is off. Reply here tomorrow and I'll diagnose.

### 8:00 PM — End of day (5 min)
- [ ] Update `ops/status.md` with today's numbers.
- [ ] Bookmark the best-performing post for reference.
- [ ] Close the laptop. No midnight scrolling for comments.

---

## Day 2

- [ ] Reply to remaining comments from day 1.
- [ ] If day 1 sales < 3: write a short X post sharing one customer quote (if any) or a reflective "what I learned shipping" post. Don't re-launch.
- [ ] If day 1 sales 3–10: post on X "sold {N} copies in 24h, thank you — here's the one thing I didn't put in the pack but should have" → share a bonus prompt. Drives second wave.
- [ ] If day 1 sales > 10: quiet victory lap. Don't flex numbers publicly — it attracts low-quality attention and competitors who'll copy you.

## Day 3–4

- [ ] Post launched drip from `marketing/drip/pack-01/week-2026-17.md` across the week.
- [ ] Keep conversations alive. DMs, replies.

## Day 6 — Last-chance

- [ ] Post X/Threads: "launch discount ends in 24h." Don't spam LinkedIn with this — it reads as desperate there.
- [ ] Post on Threads, not X if X already got the memo twice.

## Day 7 — Retro

- [ ] `ops/status.md` weekly row filled in.
- [ ] Decision: was the launch above or below expectation? Below → what's the hypothesis? Single-variable test next week.
- [ ] End discount code.

---

## Anti-patterns that will kill this launch

- **Posting everywhere at once, then panicking about each platform's metrics.** Pick the 3 posts that matter (X thread, Reddit, LinkedIn), do them well, let Threads ride.
- **Replying to comments as if you're defending the pack.** You're not. Be helpful, move on.
- **Buying an ad/boost.** Not yet. At $0 audience, paid traffic is expensive and the analytics are noisy. Wait until you have an organic baseline to compare against.
- **Launching a second product the same week.** Diffuses attention. Ship pack-02 next week or later.
- **Reading competitors' launches and trying to change yours mid-flight.** You already shipped. Hold the line.
