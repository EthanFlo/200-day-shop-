# One-Time Setup Checklist

Complete these before launching. Budget 1–2 hours. Do them in order.

## 1. Payment & storefront (30 min)

- [ ] Create a Gumroad account: https://gumroad.com/signup
- [ ] Add payout method (bank account via Stripe). International users: verify Gumroad supports your country for payouts.
- [ ] Set up your shop name and profile bio. Suggestion: **"AI Playbooks by Ethan"** or a niche-specific handle.
- [ ] Add a profile picture (use a clean logo or your photo — no AI-looking avatars, people buy from humans).

## 2. Social accounts (20 min)

You need at least **X (Twitter)** and **Reddit**. Threads and LinkedIn are bonus. Use the same handle across platforms if possible.

- [ ] X: create or repurpose an account. Bio: "AI tools & prompts that actually ship. New pack every week → [gumroad link]"
- [ ] Reddit: make sure the account is ≥7 days old and has ≥50 karma before posting in subs like r/ChatGPTPromptGenius, r/artificial, r/EntrepreneurRideAlong. Newer accounts get banned on post.
- [ ] Threads: auto-created via Instagram. Link from your X bio.
- [ ] LinkedIn (optional): best for the more "professional" packs (business prompts, enterprise use cases).

## 3. Content tooling (10 min)

- [ ] Install a scheduler (free): **Typefully** for X, **Buffer** free tier for cross-posting. Lets you queue a week of content in one sitting.
- [ ] Create a "link in bio" page if you'll have >1 product: bento.me or beacons.ai. Free.

## 4. Analytics (5 min)

- [ ] In Gumroad: go to Settings → Integrations → add UTM-ready links. You'll paste these into each social post so we can see which platform sells.
- [ ] Bookmark the Gumroad "Sales" dashboard. This is the truth. Don't chase vanity metrics on social — chase this one.

## 5. Legal housekeeping (10 min)

- [ ] Gumroad handles VAT/sales tax collection automatically. Confirm your tax settings are on.
- [ ] If you're US-based and expect >$600/yr, Gumroad issues a 1099. Use a real name + SSN/EIN.
- [ ] Skip LLC for now. Form one only if you pass $2k/month revenue — it's not necessary to start.

## 6. The "no" list (avoid these traps)

- **Do not** spam Reddit with product links. You'll be banned. The launch post template provides value first, links at the bottom.
- **Do not** use fake reviews or fake "X people just bought this" pop-ups. Gumroad will ban you and you lose all earnings.
- **Do not** buy a course on "how to make $10k/mo on Gumroad." They're selling the dream, not the business.
- **Do not** obsess over the website design week 1. Gumroad's default page converts fine. Ship products.

## When you're done

Mark a note in `ops/status.md` (create it) with the date and your Gumroad URL. Then move to launch: `marketing/pack-01-launch-content.md`.
